# AI Mock Interview Pro 🚀

**Current Version: v1.2.0**

AI Mock Interview Pro is a professional full-stack web application designed to help developers and students ace their technical interviews. Using advanced AI, the platform simulates real-world interview scenarios, provides instant feedback, and tracks progress over time.

![AI Mock Interview Pro](https://picsum.photos/seed/interview/1200/600)

## ✨ Features

- **🔐 Secure Authentication**: JWT-based login and registration system with encrypted passwords.
- **👁️ Password Visibility Toggle**: Show/hide password on login and register pages for better usability.
- **🌗 Dark / Light Mode**: Fully functional theme toggle with persistent preference via localStorage.
- **🎭 Role-Based Interviews**: Choose from various roles like Frontend, Backend, Full Stack, Java Developer, or Data Analyst.
- **⚙️ Difficulty Levels**: Practice at Easy, Medium, or Hard levels to match your experience.
- **🤖 AI-Generated Questions**: Dynamic technical questions generated by Gemini AI tailored to your specific role.
- **📊 Real-time Evaluation**: Get instant scores (0-10), detailed feedback, and suggested improvements for every answer.
- **💡 Ideal Answers**: See sample perfect answers to learn the best way to respond to tough questions.
- **📜 Interview History**: Track your growth with a persistent history of all past interview sessions and scores.
- **🎨 Modern UI/UX**: A clean, professional "Warm Organic" design built with Tailwind CSS v4 and Framer Motion.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite 6, Tailwind CSS v4, Framer Motion, Lucide React.
- **Backend**: Node.js, Express.
- **Database**: SQLite (via `better-sqlite3`) for robust local data persistence.
- **AI**: Google Gemini AI (`@google/genai`).
- **Auth**: JSON Web Tokens (JWT) & Bcrypt.js.
- **Dev Tooling**: TypeScript, tsx, Vite HMR.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Gemini AI API Key (from [Google AI Studio](https://aistudio.google.com/))

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Vansh2802/Ai-Mock-Interview-Prep.git
   cd Ai-Mock-Interview-Prep
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add:
   ```env
   GEMINI_API_KEY=your_api_key_here
   JWT_SECRET=your_jwt_secret_here
   ```

4. **Run the application**:
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.

## 🧠 How it Works

1. **Structured Prompting**: The system uses a specialized algorithm to prompt the LLM, ensuring responses are returned in a strict JSON format for programmatic handling.
2. **Semantic Evaluation**: Instead of simple keyword matching, the AI evaluates the *semantic intent* and technical accuracy of user answers.
3. **Cryptographic Security**: Passwords are hashed using the Blowfish-based Bcrypt algorithm, ensuring one-way security.
4. **State Machine**: The frontend manages the interview flow using a finite state machine to ensure a smooth, bug-free transition between questions and evaluations.
